# 🎮 Games in Python

Welcome to the **Games in Python** repository!
This project is a collection of simple and interactive games built using Python. It is designed for **beginners to learn programming concepts through fun projects**.

---

## 🚀 About the Project

This repository contains multiple mini-games developed using core Python concepts such as:

* Loops
* Conditional statements
* Functions
* Lists & Strings
* Random module

Building games is a great way to improve **logic building and problem-solving skills** while learning Python. ([GitHub][1])

---

## 🎯 Games Included

* 🧭 Treasure Hunt Game
* ✊ Rock Paper Scissors
* 🔢 FizzBuzz
* 🔐 Password Generator
* 🎲 (More games coming soon...)

---

## 🛠️ Technologies Used

* Python 3
* Standard Libraries (like `random`)

## ▶️ How to Run

1. Install Python on your system
2. Clone the repository:

   ```bash
   git clone https://github.com/ShuaibKaldane/Games-in-Python
   ```
3. Navigate to the project folder:

   ```bash
   cd Games-in-Python
   ```
4. Run any game file:

   ```bash
   python filename.py
   ```
